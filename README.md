# 🌐 Portfolio

This is my personal portfolio website, where I showcase my projects, skills, and a bit about myself as a developer.

The goal of this project is to present who I am, what I can do, and how I think as a developer.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ✨ Features

* Dynamic age calculation based on current date
* Responsive layout
* Personal information and introduction
* Clean and simple UI

## 📂 Project Structure

* `index.html` — Main page
* `style.css` — Styling
* `script.js` — Interactivity and logic

This portfolio was built as part of my learning journey, focusing on improving my front-end skills and practicing real-world development scenarios.
